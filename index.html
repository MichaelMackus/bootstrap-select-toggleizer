Select Toggleizer Convert select boxes into button group toggles View on GitHub »

Example

Usage

Create a select element:

        
  <select class="select-toggleizer">
    <option>Setting A</option>
    <option>Setting B</option>
    <option>Setting C</option>
  </select>

And toggleize it:


  $('.select-toggleizer').toggleize();

Custom Select Combo Allow custom input with select boxes View on GitHub »

Example

Usage

Create a select element:

        
  <select class="custom-select-combo">
    <option value='dog'>Dog</option>
    <option value='cat'>Cat</option>
    <option value='custom'>Custom</option>
  </select>

And customize it:


  $('.custom-select-combo').customSelectCombo();
